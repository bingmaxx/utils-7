### `Commit` 约定
- feat：新功能 feature
- fix：修补 bug
- docs：文档 documentation
- style：格式（不影响代码运行的变动）
- refactor：重构
- test：增加测试
- chore：构建过程或辅助工具的变动
- opti：优化 optimization
- non-compatibility：非兼容性修改
- breaking：破坏性更新


### 工具
- [x] 测试框架 `Mocha`

- [ ] 编译 `Babel`
